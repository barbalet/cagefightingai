Method
------
The original STL was loaded with duplicate vertex cleanup enabled, then reduced
with quadric decimation using trimesh and fast-simplification in a temporary
Python environment. The original STL, high-detail derivative, and medium-detail
derivative were removed after review; only the low, superlow, and current robot
retool variants are retained.

An earlier vertex-clustered rounding pass was rejected because it broke the
figure into ghost-like shards. The current robot retool variants avoid vertex
clustering. They start from `boxeadora-paraimprimir_low_35k.stl`, remove only
small disconnected components under 150 faces, apply mild Taubin smoothing,
preserve bounds, and use conservative quadric decimation. This keeps the figure
as coherent source components while reducing sharpness and element count. The
150-face cutoff also removes the two 126-face armpit shard components that
survived the earlier cleanup pass.
